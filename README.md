# 프로젝트 개요
* 개발 기간 : 2025.09.02 ~ 2025.09.03
* 개발 인원 : 1인

# Tech Stack
* Language : Python
* Library
    * customtkinter : 현대적인 UI 컴포넌트 및 다크 모드 구현
    * Pillow (PIL) : 이미지 처리 및 아이콘 렌더링
    * random : 컴퓨터의 랜덤 선택 로직 구현
    * urllib & xml.etree.ElementTree : REST API 호출 및 XML 데이터 파싱
    * threading : API 통신 중 UI 프리징 방지를 위한 멀티스레딩
    * math & Tkinter Canvas : 플레이어 원형 배치 및 시각적 아이콘(총, 주사기 등) 렌더링

# 핵심 기능
* 가위바위보 게임 (game.py)
    * 현대적인 UI/UX 설계
        * 다크 테마(Dark Mode) : 사용자 눈의 피로도를 낮추는 세련된 다크 블루 테마 적용
        * 반응형 버튼 : hover_color를 활용하여 버튼에 마우스를 올렸을 때의 시각적 피드백 제공
        * 이모지 활용 : ✌️, ✊, ✋ 이모지를 사용하여 직관적인 인터페이스 구축

    * 객체 지향 프로그래밍
        * RPSGameFrame 클래스 : 게임의 핵심 로직과 화면 구성을 CTkFrame으로 캡슐화하여 유지보수성과 재사용성을 높였습니다
        * RPSGameApp 클래스 : 메인 윈도우 설정 및 앱 전체의 생명 주기를 관리합니다

    * 게임 로직 및 상태 관리
        * 실시간 점수판 : '나', '무승부', '컴퓨터'의 점수를 실시간으로 업데이트 및 표시
        * 결제 판정 알고리즘 : Win-Condition 세트를 활용하여 짧고 효율적인 승패 판정 로직 구현
        * 초기화 및 종료 : 게임 중 점수를 초기화하거나 안전하게 앱을 종료하는 기능 포함

* 끝말잇기 게임 (korea_word.py)
    * 실시간 사전 데이터 검증
        * 사용자가 입력한 단어가 실제 존재하는 3음절 명사인지 API를 통해 실시간으로 확인합니다
        * 데이터의 무결성을 위해 API 응답값의 품사 태그를 분석하여 명사 여부를 판별합니다

    * 한국어 두음법칙 알고리즘 구현
        * 한국어 특유의 두음법칙(ㄹ->ㄴ, ㄴ->ㅇ 등)을 처리하기 위한 매핑 사전(DOOUM_RULES)을 구축했습니다
        * 역방향/정방향 검사 : 마지막 글자가 두음법칙 대상일 경우, 가능한 모든 초성을 계산하여 다음 단어의 시작 글자와 대조합니다

    * 비동기 처리 및 타이머 시스템
        * 멀티스레딩 : 컴퓨터가 API를 통해 다음 단어를 찾는 동안 사용자의 UI가 멈추지 않도록 threading.Thread를 사용하여 비동기 로직을 구현했습니다
        * 실시간 타이머 : after() 메서드를 활용하여 0.1초 단위로 갱신되는 프로그레스 바 형태의 시각적 타이머를 제공합니다

    * 지능형 컴퓨터 AI
        * 컴퓨터는 사용자가 낸 단어의 마지막 글자로 시작하는 모든 단어를 API에서 추출한 뒤, 그 중 랜덤하게 하나를 선택하여 응답합니다

* 마피아 게임 (mafia_game.py)
    * 시스템 제어 : 관리자 권한을 통해 경기 스케줄을 업데이트하거나(ScheduleController), 전체 회원 및 판매 현황 관리
    * 데이터 영속성 : Oracle DB와 연동하여 모든 트랜잭션 데이터를 안전하게 관리

    * 유연한 플레이어 설정
        * 4명에서 6명까지 플레이어 수 설정 가능
        * 플레이어별 이름 입력 및 랜덤 역할(마피아, 의사, 경찰, 시민) 배정

    * 시각화된 밤의 단계
        * 원형 배치 알고리즘 : 플레이어들을 캔버스 위에 원형으로 자동 배치하여 시각적 몰입감 제공
        * 인터랙티브 타겟팅 : 마우스 클릭으로 대상을 지목하며, 직업에 따라 고유한 아이콘 표시
        * 마피아 : 제거 대상에게 빨간색 총 궤적 표시
        * 의사 : 보호 대상에게 하늘색 주사기 아이콘 표시
        * 경찰 : 조사 대상에게 금색 돋보기 아이콘 표시

    * 지능형 게임 로직
        * 밤 결과 자동 산출 : 마피아의 공격과 의사의 치료를 대조하여 생존 여부 판별
        * 경찰 조사 : 지목한 대상의 정체를 실시간으로 확인
        *승리 조건 자동 체크
            * 마피아가 모든 시민을 제거하거나 시민 수와 같아질 경우 → 마피아 승리
            * 모든 마피아가 제거될 경우 → 시민 승리

    * 낮 토론 및 투표 시스템
        * 가변 토론 타이머 : 사용자가 설정한 초 단위로 토론 시간 카운트다운
        * 투표 집계 : 드롭다운 메뉴를 통한 익명 투표 및 결과 요약 제공
